# 📌 Project Overview

The **Employee Management System (EMS)** is a CRUD-based web application where users can manage employee data easily.

This project allows you to:

✅ Add Employee  
✅ View Employee List  
✅ Edit Employee Details  
✅ Delete Employee Record  

This system is created using **ADO.NET** (without Entity Framework) and uses **AJAX** so the page does not reload during CRUD operations.

---

# 🎯 Main Objective / Purpose of Project

The purpose of this project is to help developers understand:

✅ How to build ASP.NET MVC project  
✅ How to connect SQL Server using ADO.NET  
✅ How to perform CRUD operations  
✅ How to use jQuery and AJAX in MVC  
✅ How to load data dynamically in a table  
✅ How to create professional 3D UI using CSS  

This is a complete beginner-friendly project.
